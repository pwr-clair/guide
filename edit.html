가게명·지도 링크·설명을 폼에서 고치고, 아래 [guide-data.js 내려받기]를 누르세요.

받은 파일을 GitHub에 그대로 올리면 1~2분 뒤 pwr-guide.online에 반영됩니다.

설명(다국어) 규칙
  1. 한국어만 채우고 English·日本語·中文은 비워도 됩니다 → 그 언어에서는 영어(없으면 한국어)로 표시돼요.
  2. 번역을 넣고 싶은데 어렵다면, 한국어만 쓰고 클로드에게 “나머지 언어 채워줘”라고 하세요.
  3. 설명을 아예 없애려면 한국어까지 4칸 모두 비우면 됩니다(설명 줄이 사라짐).
  4. 지도 링크(URL)를 비우면 누를 수 없는 정보 카드가 됩니다.
받은 파일을 GitHub에 올리는 법
  1. github.com/pwr-clair/guide 접속 → guide-data.js 클릭
  2. 연필(✏️) 아이콘 옆의 … 또는 Upload files 로 방금 받은 guide-data.js를 덮어쓰기
  3. (또는 연필로 열고 전체 선택→붙여넣기) → 아래 Commit changes
  4. 1~2분 뒤 pwr-guide.online 에서 확인
